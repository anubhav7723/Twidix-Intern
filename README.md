# Twidix Summer Internship Program 2026

This repository contains the projects and assignments completed during the **Twidix Summer Internship Program 2026**, covering AI, automation, analytics, and business-oriented solutions. Each task was designed to provide hands-on experience with real-world technologies and problem-solving.

## 📋 Internship Tasks

### 🤖 AI Chatbot & Virtual Assistant

Designed and developed an **AI-powered Medical Chatbot & Virtual Assistant** capable of understanding natural language queries and providing intelligent, context-aware responses. The project utilizes **Retrieval-Augmented Generation (RAG)** to retrieve relevant medical information from a vector database before generating responses, improving both accuracy and reliability compared to a standalone LLM.

The chatbot integrates **LangChain** for orchestration, **Pinecone** as the vector database, **Hugging Face Embeddings** for semantic search, **Ollama (Llama 2)** for domain-specific medical responses, and **Groq Llama 3** for handling general conversational queries. A user-friendly **Flask** web interface enables seamless interaction, allowing users to ask health-related questions and receive concise, context-aware answers in real time.

#### Key Features
- AI-powered conversational virtual assistant
- Retrieval-Augmented Generation (RAG) pipeline
- Semantic search using vector embeddings
- Medical knowledge retrieval from PDF documents
- Context-aware response generation using LLMs
- General conversation support with Groq Llama 3
- Interactive Flask-based web interface
- Fast, accurate, and scalable information retrieval

**Tech Stack:** Python • Flask • LangChain • Pinecone • Hugging Face Embeddings • Ollama (Llama 2) • Groq API • RAG • Vector Database
